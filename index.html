FEATURED PROJECTS

DocuMind is an agentic, AI-powered document assistant that lets users upload unstructured documents and ask natural-language questions, returning grounded answers with relevant source context. It's built on a Retrieval-Augmented Generation (RAG) pipeline that chunks documents, retrieves the most relevant passages, and feeds them to the LLM so responses stay accurate and reduce hallucination. The system uses the Groq API for fast, low-latency inference, with engineered prompts that handle retrieval, summarization, and Q&A as distinct tasks. A Flask backend manages document ingestion, retrieval logic, and the API endpoints connecting the model to the interface, while a React frontend handles document upload and an interactive chat experience — making document interaction faster and more intuitive than manual searching.

This machine learning project predicts customer churn and surfaces retention signals that inform pricing and engagement decisions. The work began with data cleaning and exploratory data analysis to understand churn drivers, handle missing values, and identify patterns across customer segments, followed by feature engineering from behavioral and account data to strengthen the predictive signal. Random Forest and XGBoost classifiers were trained and compared, with hyperparameter tuning and AUC-ROC evaluation producing 80%+ AUC-ROC. Beyond the score, feature importance was interpreted to translate model output into actionable retention and pricing recommendations.

The Sales Analytics Dashboard is an end-to-end reporting tool built to track core KPIs, monitor trends, and support faster, data-backed decision-making. Python and Pandas were used to clean, aggregate, and transform raw sales data into a consistent, analysis-ready structure, while SQL queries extracted and joined records across sources to compute metrics like revenue, growth, and segment performance. Clear, interactive Tableau visualizations highlight trends, outliers, and KPI movement for non-technical stakeholders, simplifying recurring reporting so business users can self-serve insights instead of requesting one-off analyses.

NFT Theft Tracker is a full-stack system designed to monitor, record, and investigate NFT theft cases using structured data workflows and SQL analytics. At its core is a normalized MySQL database of 16 tables, supported by 5 views, 4 stored procedures, and 4 triggers that enforce data integrity and automate logic, with over 3,600 records loaded and managed through ETL workflows. A Node.js backend exposes 8 REST API endpoints serving theft case data, investigation status, and analytics, and a React dashboard lets users search, filter, and visualize cases — turning raw database records into an investigator-friendly interface.
